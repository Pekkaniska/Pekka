Обработка "Загрузка данных Dynafleet"

Назначение

Эта обработка позволяет производить загрузку данных местоположения ТС, скорости ТС, общего количества потребленного топлива, текущем уровне топлива в баке ТС и текущего значения одометра. Данные загружаются с сайта Dynafleet и записываются в регистр сведений "Местоположение по GPS".

Обработка может работать в 2х режимах:

1. При выбранном переключателе "Загрузка с сайта" данные о ТС будут получены с сайта Dynafleet. При этом будут получены сведения от последнего момента входа в систему до текущей даты. Если в случае загрузки данных будут найдены ТС, идентификатор которых не совпадают ни с одним имеющимся в справочнике "ТС", то пользователю будет предложено сохранить все данные во внешний файл в формате ".xml" для последующей обработки, иначе все данные будут утеряны;

2. При выбранном переключателе "Загрузка из файла" данные о ТС будут получены из ранее сохраненного файла в формате ".xml".

Формы

  • Форма. Основная форма. На форме расположены следующие элементы:
    1. Переключатель "Загрузка с сайта/Загрузка из файла";
    2. При выбранном значении переключателя "Загрузка с сайта" на форме есть реквизиты "Логин" и "Пароль" - тип Строка. Эти реквизиты - логин и пароль пользователя на сайте Dynafleet соответсвенно;
    3. При выбранном значении переключателя "Загрузка из файла" на форме есть реквизит "Выбор файла", связанный с внешним файлом; 
    4. Реквизит "Детализация загрузки данных" - тип Число. Указывает с каким интервалом данные будт записываться в регистр.

Способ вызова

Вызывается из меню "Работа ТС --> Сервис --> Загрузка данных Dynafleet".
