Обработка "Помощник планирования работы ТС"

Назначение

Обработка предназначена для заполнения планов работы ТС по данным заказов на ТС или на основании выработки ТС за предыдущий период.

Использование

При заполнении может быть использована одна из 5 стратегий планирования:

  • По заказам на ТС. Используются данные табличных частей "Заказы" заказов на ТС: номенклатура груза, параметр выработки и количество по параметру выработки.
  • По выработке ТС. Используются данные по выработке ТС (путевые листы) за предыдущий период.
  • По проданным билетам. Используются данные по выручке по путевым листам за предыдущий период.
  • По заказам и по выработке ТС. Используются данные и по заказам на ТС, и по выработке ТС.
  • По заказам и по проданным билетам. Используются данные и по заказам на ТС, и по проданным билетам.

Способ вызова

Вызывается из меню "Заполнить --> Заполнить план" документа Планирование работы ТС.
