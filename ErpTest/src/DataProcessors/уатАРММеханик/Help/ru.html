Обработка "АРМ механика"

Обработка "АРМ Механика" предназначена для автоматизации рабочего места механика. Основное назначение обработки -  анализ состояния транспортных средств (ТС), приближения ТО, работа с агрегатами (шинами) ТС.  

Вся информация функционально сгруппирована по закладкам.

 Закладка "Транспорт".

 Закладка предназначена для подбора ТС в документ "Операции с агрегатами". В списке "Транспортные средства" отображены ТС организации и их признаки – приближение срока тех-обслуживания (ТО) и просрочка документов.  С помощью кнопки  "Отбор ТС (Подошел срок ТО)"  можно включить отбор ТС по сроку приближающегося ТО, то есть в списке останутся только те Тс у которых подошел плановый срок ТО.  

Двойной щелчок мыши на ТС в списке "Транспортные средства" осуществляет подбор его в поле "ТС" на закладке "Операции с агрегатами".

В нижней части страницы отображены следующие информационные данные (сгруппированы по страницам):

·         Подошедшие ТО (отображает дополнительную информацию по ТО для выбранного ТС, например, вид ТО, параметры выработки, плановые сроки ТО и т.п.);

·         Установленные агрегаты (информация о установленных на ТС агрегатах – шинах и аккумуляторах);

·         История операций с агрегатами, установленными на выбранное ТС.

 Закладка "Агрегаты".

Закладка предназначена для работы с агрегатами ТС (шины). Левая верхняя таблица содержит информацию о моделях шин. Ниже выводятся свободные агрегаты (не установленные на ТС или снятые с ТС) по выбранной модели шин с указанием общих характеристик и серийного номера шины. Закладка "Отбор" позволяет установить фильтрацию списка моделей по различным характеристикам, таким как, модель, сезонность и т.п.

Для упрощения работы с установкой/снятием шин разработана возможность установки/снятия шины используя механизм drag&drop("перетащил и бросил") и графический макет автомобиля. При выборе ТС автоматически загружается его графический макет (заполняется в справочнике "Модели ТС"). Установить шину на ТС можно перетащив мышью любую (необходимую) шину из таблицы остатков шин и бросив ее в таблицу "Установка шин" (при этом нужно указать место установки в открывшемся окне). В таблице "Установленные шины" отображаются уже установленные на ТС шины. Снять шину можно перетащив требуемую шину из таблицы "Установленные шины" на таблицу "Перенос шин". Так же эта таблица служит буфером при перемещении шин между местами установки.

Кнопка "Создать" формирует  документ "Операции с агрегатами", автоматически заполняя все данные по проведенным операциям с шинами. Кнопка "Очистить" восстанавливает макет в актуальное состояние (загружает заново данные по установленным шина и очищает вспомогательные таблицы).
