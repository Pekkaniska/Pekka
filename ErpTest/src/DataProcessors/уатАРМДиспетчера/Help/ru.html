Обработка "АРМ диспетчера"

Обработка "АРМ Диспетчера" предназначена для автоматизации рабочего места диспетчера. Основное назначение обработки -  помощь в отработке заказов покупателей и формирования заданий на перевозку транспортными средствами (ТС) организации.  

Вся информация функционально сгруппирована по закладкам.

 Закладка "Заказы на ТС".

 Закладка предназначена для подбора заказов подлежащих развозу. Для удобства работы со списком заказов, есть возможность фильтровать (устанавливать отбор) это список по следующим критериям:

·          Все (показываются все без исключения заказы);

·          Текущие (заказы, по которым дата отгрузки совпадает с текущей датой);

·          Показать по регионам (заказы, адрес доставки для которых совпадет с выбранным значением региона в списке "Регионы доставки").

Двойной щелчок мыши на заказе в списке "Заказы" осуществляет подбор его в список "Выбранных заказов". В этом списке показывается оценочная характеристика суммарного объема и веса груза на основании информации о номенклатуре выбранных заказов.

 Оценка объема и веса груза используется для отбора транспортных средств по данным характеристикам. Для этого используется кнопка "Показать подходящие ТС". Так же, при включении кнопки выполняется отбор по требованию к ТС (тип и модель ТС), указанному в заказах. Чтобы отключить отбор в списке транспортных средств (если он включен) необходимо нажать кнопку "Отключить отбор".

 Закладка "Маршрутный лист".

 Закладка предназначена для заполнения документов "Маршрутный лист".  Данные закладки могут быть заполнены как вручную, так и автоматически. Автоматическое заполнение производится данными закладки "Заказы на ТС". Все заказы из списка "Выбранные заказы" автоматически перенесутся в табличную часть маршрутного листа. Двойной щелчок мышки на ТС в списке "Транспортные средства" заполнит его в маршрутный лист. 

Кнопка "Создать" формирует  документы, заполняя их данными этой страницы, при этом учитываются параметры группы "Создаваемые документы".  Таким образом, можно сформировать не только "Маршрутный лист", а и "Путевой лист", указать необходимость печати создаваемых документов и количество экземпляров при печати, открыть формы создаваемых документов и очистить поля закладки "Задание". Кнопка "Очистить" очищает поля закладки "Задание".

 Закладка "Маршрутные листы".

 Закладка отображает журнал "Маршрутные листы", и связанные с ним данные – список заказов покупателей, которые вошли в задание, и список путевых листов введенных на основании задания.

 Закладка "Адреса доставки".

 Закладка предназначена для работы с информацией об адресах доставки. Для этого в решении используется дополнительный регистр сведений, который позволяет группировать адреса доставки грузов но регионам и тем самым расширяет возможности типового регистра  "Контактная информация".

Для синхронизации данных с типовым регистром  "Контактная информация" предусмотрена кнопка "Обновить данными из контактов", расположенной на панели инструментов, либо в контекстном меню. Если в дереве "Контрагенты" выбран контрагент, то обновятся его данные, если группа, то все контрагенты этой группы. При обновлении нужно указать те виды контактной информации, которые подлежат синхронизации и/или обновлению.
