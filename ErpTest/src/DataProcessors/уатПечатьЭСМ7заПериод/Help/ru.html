Обработка "Печать ЭСМ-7 за период"

Назначение

Обработка предназначена для печати формы ЭСМ-7 за указанный период. Данные собираются из регистра Выработка по стоимости с отбором по параметрам выработки "Время в работе" и "Время спец. работы" (стоимость услуг, рассчитанная по другим параметрам выработки в ТТД, учитываться не будет).

Для печати необходимо указать организацию, транспортное средство (или объект строительства в зависимости от режима отбора) и дату выписки, а также список объектов строительства (или список ТС в зависимости от режима отбора). 

Режимы отбора по ТС и объектам строительства:

  1. Список "Объекты строительства" заполняется динамически при изменении реквизитов "Организация", "ТС", "Период с ", "Период по", "Контрагент".
  2. Список "ТС" заполняется динамически при изменении реквизитов "Организация", "Объект строительства", "Период с ", "Период по", "Контрагент".

По желанию можно установить отбор по периоду и контргагенту.

Если для одного объекта строительства будет получено более 7 строк с наименованиями работ, то в этом случае будет сформирован отдельный акт с указанными работами.

Способ вызова

Вызывается из меню "Работа ТС --> Печать ЭСМ-7 за период"
