Обработка "Формирование узлов объектов эксплуатации"

Назначение

Обработка предназначена для автоматического переноса данных из справочника "Серии номенклатуры" в справочник "Объекты эксплуатации".

Обработка позволяет в автоматическом режиме создать элементы справочника "Узлы объектов эксплуатации", заполнить необходимые реквизиты и создать записи об установке агрегата на транспортное средство. На форме обработки расположены колонки: "Агрегат (серия номенклатуры)" и "Агрегат (узел объекта эксплуатации)". В левой колонке отображаются агрегаты из справочника "Серии номенклатуры", в правой колонке - из справочника "Узлы объектов эксплуатации". При нажатии на кнопку "Создать узлы" для всех строк, где не заполнено значение в колонке "Узел объекта эксплуатации" выполняется проверка установки данного агрегата на какое-либо ТС. Если агрегат в данный момент установлен и для транспортного средства указан связанный объект эксплуатации, то создается и заполняется новый элемент справочника "Узлы объектов эксплуатации". При нажатии на кнопку "Обновить регистры" выполняется автоматическая замена в объектах конфигурации (регистр сведений "Агрегаты ТС", регистр накопления "Износ пробег шин", регистр накопления "Выработка агрегатов") элемента из справочника "Серии номенклатуры" на элемент из справочника "Узлы объектов эксплуатации".
