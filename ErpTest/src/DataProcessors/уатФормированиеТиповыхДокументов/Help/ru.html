Внутреннее потребление товаров

  • В колонке "Значение" дерева настроек на первой странице по строкам формирования общих документов можно ввести Дату общего документа.
    Общий документ - это вариант выгрузки всех документов, когда документы одного вида перегружаются в один стандартный документ;
  • На закладке "Отбор" можно указать значения отбора обрабатываемых документов. Если значение не заполнено, то будут обработаны все документы;
  • Для каждой заправки за наличные создается пара документов: "Приобретение товаров", которое попадает в табличную часть "Авансового отчета";
  • В создаваемом типовом документе "Оприходование излишков товаров" заполнение цен производится аналогично заполнению по кнопке "Заполнить цены / По себестоимости" командной панели табличной части "Товары". То есть, по данным документов "Расчет себестоимости товаров" и регистра сведений "Стоимость товаров";
  • В создаваемом типовом документе "Возврат товаров поставщику" заполнение цен производится аналогично заполнению по кнопке "Заполнить / Заполнить документы закупки и цены" командной панели табличной части "Товары". То есть, по данным документов "Приобретение товаров и услуг" и "Корректировка поступления";
  • При создании документа "Перемещение товаров" на основании "Операции с агрегатами" склад-отправитель и склад-получатель документа УАТ используются для заполнения реквизитов типового документа. Также используется значение настройки УАТ "Склад для перемещения агрегатов" (по организации или подразделению) - это склад, отражающий в типовом учете объем ГСМ, содержащийся в баках всех ТС организации или подразделения. 

    Способ вызова

    Вызывается из меню "Администрирование --> Сервис --> Формирование типовых документов".
