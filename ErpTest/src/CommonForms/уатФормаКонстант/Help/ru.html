Форма констант

Форма предназначена для установки параметров, которые являются общими для всех организаций информационной базы.

Закладка "Основные"

Закладка содержит информацию о текущей версии конфигурации.

Закладка "Валюты"

Закладка содержит настройки валюты регламентированного и управленческого учета. После начала ведения учета изменение данных валют невозможно.

Закладка "Единицы измерения"

Закладка содержит информацию об основной единице измерения веса и основной единице измерения объема.

Закладка "GPS"

Закладка содержит настройки взаимодействия с различными системами GPS мониторинга.

  • Закладка "Omnicomm". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "Omnicomm".
    • Использовать Omnicomm. Включает использование взаимодействия с системой GPS мониторинга "Omnicomm".
    • Способ работы. Позволяет указать источник данных для загрузки. Возможные значения: "ВебСервис", "БД".
    • Логин. Логин пользователя для подключения к источнику данных (веб сервису или базе данных).
    • Пароль. Пароль пользователя для подключения к источнику данных (веб сервису или базе данных).
    • Адрес сервиса. Адрес веб сервиса системы мониторинга. (Доступно только для способа работы "ВебСервис").
    • Подключение. Способ подключения к базе данных. Возможные значения: "Стандартное подключение", "PIPE-подключение", "DSN-подключение". (Доступно только для способа работы "ВебСервис")
    • Сервер. Адрес сервера базы данных. (Доступно только для способа работы "ВебСервис").
    • Имя базы. Имя базы данных на сервере. (Доступно только для способа работы "ВебСервис").
    • Драйвер. Имя драйвера для подключения к серверу. (Доступно только для способа работы "ВебСервис").
    • Имя PIPE. Наименвание именованного канала для работы с сервером. (Доступно только для способа работы "ВебСервис").
    • Группа "Параметры регламентного задания "Загрузка данных"":
      • Загрузить дополнительные сведения. При работе регламентного задания "Загрузка данных omnicomm" будут загружены дополнительные сведения по указанным в таблице ТС транспортным средствам.
      • Загрузить пробег и расход ГСМ. При работе регламентного задания "Загрузка данных omnicomm" будет загружен пробег и расход топлива по указанным в таблице ТС транспортным средствам.
      • Интервал загрузки. Интервал времени за который будут загружены данные.
      • Таблица ТС. Позволяет указать список транспортных средств, по которым необходимо загружать данные регламентным заданием "Загрузка данных omnicomm".
  • Закладка "Dynafleet". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "Dynafleet".
    • Использовать Dynafleet. Включает использование взаимодействия с системой GPS мониторинга "Dynafleet".
    • Логин. Логин пользователя для подключения к веб сервису.
    • Пароль. Пароль пользователя для подключения к веб сервису.
    • Адрес сервиса. Адрес веб сервиса системы мониторинга.
    • Детализация загрузки данных. Позволяет указать с каким интервалом (минут) будут загружаться данные из системы "Dynafleet".
    • Группа "Параметры регламентного задания "Загрузка данных"":
      • Загружать с сайта. При загрузке регламентным заданием "Загрузка данных dynafleet" данные будут получены из системы GPS мониторинга "Dynafleet".
      • Загружать из файла. При загрузке регламентным заданием "Загрузка данных dynafleet" данные будут получены из файла, указанного в настройке "Имя файла".
      • Имя файла. Путь к файлу, содержащему данные для загрузки.
  • Закладка "ЦСМ". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "ЦСМ".
    • Использовать ЦСМ. Включает использование взаимодействия с системой GPS мониторинга "ЦСМ".
  • Закладка "Position report". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "Position report".
    • Использовать Position report. Включает использование взаимодействия с системой GPS мониторинга "Position report".
    • Логин. Логин пользователя для подключения к веб сервису.

    • Адрес сервиса. Адрес веб сервиса системы мониторинга.
    • Пространство имен. Пространство имен веб сервиса.
  • Закладка "Автотрекер". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "Автотрекер".
    • Использовать Автотрекер. Включает использование взаимодействия с системой GPS мониторинга "Автотрекер".
    • Логин. Логин пользователя для подключения к веб сервису.
    • Ключ сервиса данных. Авторизационный ключ пользователя для подключения к веб сервису данных.
    • Ключ сервиса отчетов. Авторизационный ключ пользователя для подключения к веб сервису отчетов.
    • Адрес сервиса данных. Адрес веб сервиса данных системы мониторинга.
    • Адрес сервиса отчетов.  Адрес веб сервиса отчетов системы мониторинга.
  • Закладка "СКАУТ". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "СКАУТ".

      • Интервал загрузки. Интервал времени за который будут загружены данные.
      • Таблица ТС. Позволяет указать список транспортных средств, по которым необходимо загружать данные регламентным заданием "Загрузка данных СКАУТ".
  • Закладка "АвтоГРАФ". Позволяет включить использование механизмов взаимодействия с системой GPS мониторинга "АвтоГРАФ".

Закладка "SMS"

На данной закладке находится настройка, позволяющая включить использование SMS-модуля и осуществить переход к форме его настройки.

Закладка "Прочие"

Закладка содержит настройки прочих блоков конфигурации:

  • подсистемы бюджетирования;
  • настройка выполнения регламентного задания в файловом режиме;
  • подсистемы дополнительных реквизитов и сведений;
  • подсистемы подключаемого оборудования и штрихкодирования;
  • подсистема взаимодействия с мобильным приложением "Водителя УАТ".
