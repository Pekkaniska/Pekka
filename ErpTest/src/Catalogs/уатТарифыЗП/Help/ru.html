Справочник "Тарифы зарплаты сотрудников"

Назначение

Справочник содержит информацию по тарифам, согласно которым ведется начисление зарплаты сотрудникам в Путевых листах.

Относится к подсистеме "Управление автотранспортом".

Реквизиты

  • Вид начисления. Ссылается на план вида расчетов «Начисления». Используемый вид начисления зарплаты.
  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки, по которому отбирается выработка водителей.
  • Сложный тариф. Тип значения: «Булево». Флаг "сложного" тарифа. "Сложным" считается тариф, размер которого определяется по совокупности условий, согласно таблице расчета (закладка "Настройка расчета").
  • Группировка тарифа №1-3. Ссылается на перечисление «Группировки тарифов». Вид группировки: расстояние, время работы, грузоподъемность, класс груза.
  • Тариф. Тип значения: «Число». Размер тарифа. Используется для "простого" тарифа.
  • Расчет по строкам. Тип значения: «Булево». Если флажок установлен, то будет использоваться построчный расчет зарплаты в путевом листе. Расчет выполняется по ТЧ Задание, либо по ТЧ Выработка, если в путевом листе используется режим расчета - по ТТД. Если флажок снят, то расчет ведется по итоговым показателям выработки (значения по параметрам выработки суммируются по всем строкам задания).

Табличная часть «Группировки»

  • Значение группировки 1-3. Тип значения: «Число». Значение группировки.
  • Знак группировки 1-3. Ссылается на перечисление «Знаки сравнения». Знак группировки.
  • Тариф. Тип значения: «Число». Размер тарифа в валюте упр. учета. Используется для "сложных" тарифов.

Табличная часть «Области действия»

Табличная часть предназначена для настройки фильтров действия данного тарифа (описание работы см. ниже). Строки, выделенные красным шрифтом, не будут учитываться при расчете зарплаты. Например, не будут учитываться маршруты или контрагенты, если снят флажок "Расчет по строкам".

Табличная часть «Организации»

Табличная часть предназначена для настройки фильтров действия данного тарифа по организациям (описание работы см. ниже).

Способ вызова

  • Вызывается из меню "Работа ТС --> Учет зарплаты --> Тарифы з/п".

    Особенности работы со справочником

    1. "Сложные" тарифы (группировки) устанавливаются по выполнению всех условий строк таблицы расчета. Например, если расстояние > 100 км и вес груза > 10т, тогда размер тарифа = 100, если расстояние <=100 и вес груза > 10т, тогда тариф = 75, и т.д. Чаще всего используется одна группировка, например по расстоянию, когда свыше какого то километража начисляется дополнительная сумма, а до этой границы расстояния начисления идут по обычному тарифу.
    2. Область действия тарифа - это фильтр, согласно которому тариф будет учитываться при расчете зарплаты водителя. Например, имеем область Контрагент = ЧП "Петров", Модель ТС = "МАЗ 07". Тогда по всем ремонтным / путевым листам, где указаны эти значения, данный тариф будет входить в расчет зарплаты сотрудников. Важная особенность области действия: если не указан вид области действия, то при выборе тарифов она не будет учитываться вообще. Например, если в тарифе указана область действия по модели ТС, но не указано конкретное ТС, то тариф будет учитываться для всех ТС данной модели.
    3. Организации. Список организаций, которые включаются в отбор при поиске тарифов. Фактически, Организация - та же самая область действия тарифа, только вынесена для удобства представления и работы с тарифом в отдельную таблицу.
    4. Расчет по строкам можно использовать при необходимости более точно рассчитывать зарплату по сложным тарифам. Например, есть сложный тариф, зависящий от расстояния: если расстояние > 100, тогда 50 руб, иначе - 45 руб. В ТЧ Задание находятся 2 строки, в первой строке указан пробег 110 км, во второй - 40 (всего 150). Если флаг "Расчет по строкам" снят, то начисление будет рассчитано так: (110+40)*50 = 7500. Если флаг установлен: 110*50 + 40*45 = 7300.
