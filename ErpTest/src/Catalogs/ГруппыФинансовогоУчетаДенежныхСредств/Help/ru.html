Группы настроек финансового учета денежных средств

Справочник предназначен для классификации объектов в системе по правилам отражения в бухгалтерском (финансовом) учете.

Варианты использования Группы настроек финансового учета денежных средств

  • Для объединения движений по банковским счетам и кассам, отражаемых на одинаковых счетах учета.
  • Для получения счетов международного финансового учета при формировании проводок по документам движения денежных средств.
  • В шаблонах проводок при настройке уточнения счетов учета.
