Справочник "АЗС"

Назначение

В справочнике содержится информация о всех АЗС, на которых осуществляются заправки ГСМ. В качестве АЗС может быть указан собственный склад, либо контрагент.

В таблице ГСМ указывается соответствие между ГСМ из отчетов процессинговых центров и ГСМ, используемым в решении УАТ. Таблица соответствий используется при загрузке данных из отчетов процессинговых центров.

Относится к подсистеме "Учет ГСМ".

Реквизиты

  • Наименование. Тип значения «Строка». Наименование АЗС.
  • В зависимости от переключателя Внешний поставщик/Внутренний склад реквизит Контрагент / склад ссылается на справочник «Контрагенты» - указывается сторонняя АЗС, либо на справочник «Склады» - указывается собственный склад.

Табличная часть «Таблица соответствия ГСМ».

  • ГСМ из файла. Тип значения «Строка». Название ГСМ из файла.
  • ГСМ. Ссылается на справочник «Номенклатура». Номенклатура-ГСМ для соответствия.

Табличная часть «Цены контрагентов».

Связана с регистром сведений «Цены номенклатуры контрагентов».

  • Период. Тип значения «Дата». Дата установки цены номенклатуры.
  • ГСМ. Ссылается на справочник «Номенклатура». Номенклатура контрагента.
  • Цена. Тип значения «Число». Цена указанной номенклатуры.
  • Валюта. Ссылается на справочник «Валюты». Валюта, в которой указана цена номенклатуры.

Способ вызова

Вызывается из меню "ГСМ --> АЗС"
