Справочник "Модели транспортных средств и оборудования"

Назначение

Справочник содержит список используемых моделей транспортных средств и оборудования и задает их характеристики, не зависящие от конкретного экземпляра.

Относится к подсистеме "Работа ТС".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Иерархический: группы и элементы.

Реквизиты

  • Вид модели ТС. Ссылается на перечисление «Виды моделей ТС». Вид модели ТС: автотранспорт, прицеп, оборудование.
  • Наличие спидометра. Тип значения: «Булево». Флаг наличия спидометра.
  • Наличие счетчика моточасов. Тип значения: «Булево». Флаг наличия счетчика моточасов.
  • Тип ТС. Ссылается на справочник «Типы ТС». Тип ТС.
  • Категория по классификатору ЕЭК. Ссылается на перечисление «Категории АТС по классификатору ЕЭК ООН». Категория оп классификатору ЕЭК.
  • Основное топливо. Ссылается на справочник «Номенклатура». Основное топливо.
  • Дополнительное топливо. Ссылается на справочник «Номенклатура». Дополнительное топливо.
  • Наличие топливного бака. Тип значения: «Булево». Флаг наличие топливного бака.
  • Объем бака. Тип значения: «Число». Объем бака, л.
  • Максимальная скорость. Тип значения: «Число». Максимальная скорость.
  • Собственный вес. Тип значения: «Число». Собственный вес, т.
  • Грузоподъемность. Тип значения: «Число». Грузоподъемность, т.
  • Объем кузова. Тип значения: «Число». Объем кузова, м3.
  • Количество осей. Тип значения: «Число». Количество осей.
  • Количество колес. Тип значения: «Число». Количество колес.
  • Наличие топливного бака. Тип значения: «Булево». Флаг наличие топливного бака.
  • Тип двигателя. Ссылается на перечисление «Типы двигателей». Тип двигателя: бензиновый, дизельный и т.п. 
  • Модель двигателя. Тип значения: «Строка». Модель двигателя.
  • Мощность двигателя (квт). Тип значения: «Число». Мощность двигателя в квт.
  • Мощность двигатель (л.с.). Тип значения: «Число». Мощность двигатель в л.с.
  • Рабочий объем двигателя. Тип значения: «Число». Рабочий объем двигателя, см3.
  • Длина. Тип значения: «Число». Длина, мм.
  • Высота. Тип значения: «Число». Высота, мм.
  • Ширина. Тип значения: «Число». Ширина, мм.

Табличная часть «Нормы ТО».

  • Вид обслуживания. Ссылается на справочник «Виды обслуживания». Вид обслуживания/ремонта ТС. Например, ТО1, ТО2 и т.п.
  • Периодичность обслуживания. Ссылается на перечисления «Периодичность». Плановая периодичность обслуживания. В списках ТС подошедшие ТО отображаются значком .
  • Количество периодов. Тип значения: «Число». Количество периодов. Например, для планирования ТО один раз в 2 месяца необходимо указать периодичность обслуживания = месяц, а количество периодов = 2.
  • Параметр выработки. Тип значения: «Число». Если указан параметр выработки, то планирование ТО будет осуществляться по нему. Иначе только по периодичности обслуживания.
  • Значение по параметру выработки. Тип значения: «Число». Значение по параметру выработки для планирования обслуживания. Например, чтобы запланировать ТО на каждые 10000 км пробега нужно указать параметр выработки = общий пробег, и значение = 10000.
  • Плановый. Тип значения: «Булево». Флаг планирования обслуживания.
  • Допуск. Тип значения: «Число». Значение по параметру выработки - максимально допустимая задержка по выполнению ремонта. Например, период 60 дней, а допуск 5 дней. Тогда ремонт следует произвести по 65-й день.
  • Стоимость. Тип значения: «Число». Стоимость обслуживания плановая.
  • Валюта. Ссылается на справочник «Валюты». Валюта стоимости обслуживания.

Таблица «Нормы расхода ГСМ».

Здесь указаны нормы расхода ГСМ для текущей модели ТС или оборудования. Нормы расхода содержатся в регистре Нормы расхода ГСМ.

Таблица «Нормы расхода ТЖ».

Здесь указаны нормы расхода технических жидкостей для текущей модели ТС или оборудования. Нормы расхода содержатся в регистре Нормы расхода технических жидкостей.

Закладка «Макет».

Макет (рисунок) текущей модели ТС.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма выбора. Основная форма выбора. Используется для отображения списка выбора элементов.

Способ вызова

Вызывается из меню "Основные данные --> Транспортные средства, оборудование --> Модели ТС и оборудования".
