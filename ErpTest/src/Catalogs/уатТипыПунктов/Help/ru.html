Справочник "Типы пунктов"

Назначение

Справочник содержит список типов пунктов назначения.

Параметры

Длина наименования: 50.

Реквизиты

  • Стоянка. Время стоянки в пункте, имеющем данный тип.

Табличные части

  • Разрешенные типы ТС. Предназначена для указания типов ТС, которым разрешено посещать данный тип пункта. Содержит реквизиты:
    • Тип ТС. Тип транспортного средства.
    • Стоянка. Время стоянки по данному типу транспортного средства.
