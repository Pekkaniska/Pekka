Справочник "Модели агрегатов"

Назначение

Справочник предназначен для классификации агрегатов (шины, аккумуляторы, аптечки, огнетушители) по их техническим характеристикам.

Для модели необходимо указать тип агрегата. В зависимости от типа становятся доступны дополнительные параметры.

Для шин дополнительно указываются:

  • Размерные характеристики шин;
  • Индекс скорости шин;
  • Индекс нагрузки шин;
  • Тип конструкции шин;
  • Норма пробега - норма пробега в км до полного износа;
  • Норма затрат, % от стоимости на 1000 км - затраты на шины в % от стоимости. 

Для аккумуляторов дополнительно указываются:

  • Емкость - емкость аккумулятора;
  • Срок службы - срок службы аккумулятора;

Относится к подсистеме "Учет агрегатов".

Реквизиты

  • Наименование.  Тип значения: «Строка».  
  • Код.  Тип значения: «Строка».
  • Тип агрегата. Ссылается на справочник «Типы агрегата».
  • Производитель.  Тип значения: «Строка».  Производитель агрегата. 
  • Модель.  Тип значения: «Строка».  Модель агрегата строкой.
  • Ширина.  Ссылается на справочник «Ширина шин». Ширина шины.   
  • Высота.  Ссылается на справочник «Высота шин». Высота шины.
  • Диаметр.  Ссылается на справочник «Диаметры шин». Диаметр шины.
  • Тип конструкции.  Тип значения: «Перечисление». Тип конструкции шины.
  • Индекс нагрузки.  Ссылается на справочник «Индексы нагрузки». Индекс нагрузки шины. 
  • Индекс скорости.  Ссылается на справочник «Индексы скорости». Индекс скорости шины. 
  • Дополнительная информация.  Тип значения: «Строка». Любая дополнительная информация о шине.
  • Норма пробега, км.  Тип значения: «Число». 
  • Норма затрат, % от стоимости на 1000 км.  Тип значения: «Число». 
  • Емкость.  Тип значения: «Число». Емкость аккумулятора. 
  • Срок службы, месяцы.  Тип значения: «Число». Срок службы агрегата, месяцев.  

Способ вызова

Вызывается из меню "Учет агрегатов --> Модели агрегатов".
