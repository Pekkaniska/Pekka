Справочник "Группы пользователей для согласования"

Назначение

Справочник содержит информацию о пользователях, которые будут автоматом подставляться как исполнители в бизнес-процессы "Согласование".

Подстановка пользователей в новый бизнес-процесс будет происходить при записи документов "Ввод в эксплуатацию", "Выбытие ТС и оборудования", "Перемещение ТС и оборудования", "Установка прейскурантов ТС", "Установка тарифов зарплаты сотрудников".

Относится к подсистеме "Согласования".

Способ вызова

Вызывается из раздела "Администрирование" -> "Визирование документов -> Группы пользователей для согласования"
