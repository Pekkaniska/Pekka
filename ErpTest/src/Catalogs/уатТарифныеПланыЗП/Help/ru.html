Справочник "Тарифные планы начисления зарплаты сотрудникам"

Назначение

Справочник предназначен для объединения различных тарифов зарплаты сотрудников. Используется в карточке сотрудника при назначении тарифов. Часто используется для установки планов тарифов на некоторый период вперед, например, "Тарифный план №123 на 2008г.". Также может использоваться при использовании сложных вариантов тарификации, когда возможностей одного тарифа для этого недостаточно.

Относится к подсистеме "Работа ТС".

Табличная часть «Тарифы».

  • Тариф. Ссылается на справочник «Тарифы зарплаты сотрудников». Тариф начисления зарплаты сотрудникам/водителям.

Способ вызова

Вызывается из меню "Работа ТС --> Тарифные планы начисления з/п".
