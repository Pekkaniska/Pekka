Сценарии планирования проектов строительства

Справочник Сценарии планирования проектов строительства предназначен для хранения вариантов плана-графика объекта строительства. Каждый сценарий относится к определенному виду: рабочему, базовому или прочему. Для каждого объекта строительства может быть только один рабочий сценарий.

В карточке элемента вводятся следующие данные:

  •  Наименование - указывается наименование сценария планирования.
  •  Владелец - указывается объект строительства, который строится по данному сценарию.
  • Вид сценария -  указывается вид сценария планирования, который может принимать значение «Базовый», «Рабочий», «Прочий».
  • Начало строительства - дата начала строительства.
  • Окончание строительства - дата окончания строительства.
  • Флаг Загружен из MSProject - установка флага означает, что данный сценарий планирования проекта строительства загружен из документа MSProject.
  • Дата загрузки из MS Project - дата загрузки сценария планирования проекта строительства из документа MSProject.

 

По кнопке Открыть работы на панели действий отображается форма списка справочника Строительные работы.
