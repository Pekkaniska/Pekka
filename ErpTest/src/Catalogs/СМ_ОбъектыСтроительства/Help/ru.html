Объекты строительства

Справочник Объекты строительства предназначен для хранения списка объектов строительства. Справочник содержит список наименований объектов строительства.

Справочник содержит иерархию групп и элементов.

В карточке элемента вводятся следующие данные:

  • Наименование;
  • Заказчик - заказчик строительства, выбор его из справочника Контрагенты.
