Справочник "Параметры выработки (ТС и водителей)"

Назначение

Справочник содержит список параметров выработки, по которым производится расчет в путевых листах. Справочник содержит перечень предопределенных элементов.

Относится к подсистеме "Работа ТС".

Реквизиты

  • Наименование для документов. Тип значения: «Строка». Используется для заполнения счетов и актов.
  • Показатель наработки. Используется для взаимосвязи с объектами типовой конфигурации и формирования ТД.
  • Временной. Тип значения: «Булево». Признак того, что параметр выработки является временным, т.е его значение измеряется в часах.
  • Выработка ТС и оборудования в Путевых листах. Тип значения: «Булево». Флаг действия на ТС.
  • Выработка водителей в Путевых листах. Тип значения: «Булево». Флаг действия на сотрудников/водителей.
  • Для строк задания с одинаковым номером этапа значение выработки (доступно для параметров выработки Вес, Объем, Количество ездок, пользовательских):
    • Суммировать. Если в путевом листе в табличной части "Задание" используется нумерация этапов, то в рамках одного этапа выработка по данному параметру будет суммироваться.
    • Выбирать максимальное. Если в путевом листе в табличной части "Задание" используется нумерация этапов, то в рамках одного этапа выработка будет выбрана из строки с максимальным значением выработки по данному параметру.
  • Использовать в тарифах на транспортные услуги. Тип значения: «Булево». Флаг использования в тарифах на услуги при расчете стоимости услуг.
  • Использовать в планировании. Тип значения: «Булево». Флаг использования в документе "Планирование работы ТС".

Способ вызова

Вызывается из меню "Работа ТС --> Справочники --> Параметры выработки ТС и водителей".
