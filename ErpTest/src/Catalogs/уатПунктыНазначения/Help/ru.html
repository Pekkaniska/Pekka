Справочник "Пункты назначения"

Назначение

Справочник содержит список пунктов назначения, по которым может осуществляться работа транспортных средств. В качестве пунктов назначения могут выступать населенные пункты, ЖД станции, адреса, стоянки такси и автобусов и т.д.

Относится к подсистеме "Работа ТС".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 100.

Иерархический: группы и элементы.

Реквизиты

  • Контрагент. Ссылается на справочник «Контрагенты». Контрагент, соответствующий пункту назначения.
  • Тип пункта. Ссылается на перечисление «Типы пунктов назначения». Тип населенного пункта: город, ПГТ, деревня и т.п.
  • Время доступности с / по. Тип значения «Дата». Временной интервал доступности пункта. Подставляется по умолчанию в документы.
  • Стоянка. Тип значения «Число». Продолжительность стоянки по умолчанию.
  • Адрес. Тип значения «Строка». Фактический адрес. Имеется возможность выбора адреса из Адресного классификатора.
  • Страна. Ссылается на справочник «Классификатор стран мира». Фактический адрес.
  • Регион. Ссылается на справочник «Регионы». 
  • Географическое местоположение (в программе Лат и Лон). Тип значения «Число». Широта и долгота пункта. Используется в заказе на ТС.

Способ вызова

Вызывается из меню "Работа ТС --> Справочники --> Пункты назначения".
