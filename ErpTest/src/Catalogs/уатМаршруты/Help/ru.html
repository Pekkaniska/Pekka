Справочник "Маршруты"

Назначение

Справочник содержит перечень маршрутов, по которым могут выполняться рейсы. Используется в документе "Путевой лист".

Относится к подсистеме "Работа ТС".

Реквизиты

  • Номер маршрута. Тип значения: «Строка». Дополнительный номер или код маршрута.
  • Расстояние. Тип значения: «Число». Общее расстояние между всеми пунктами назначения, км.
  • Время в пути. Тип значения: «Число». Плановое время в пути в формате ч:мин
  • Нулевой пробег. Тип значения: «Число». Пробег ТС от гаража до выхода на маршрут и обратно.
  • Представление маршрута. Переключатель. Список этапов маршрута может быть представлен как в виде одиночных пунктов (что, например, удобно для пассажирских перевозок) так и в виде пар пунктов следования. Эта информация используется при заполнении расписаний движения по маршрутам.

Табличная часть «Пункты следования».

  • Пункт отправления. Ссылается на справочник «Пункты назначения». Пункт отправления.
  • Пункт прибытия. Ссылается на справочник «Пункты назначения». Пункт прибытия.
  • Расстояние. Тип значения: «Число». Расстояние между пунктами, км.
  • Время в пути. Тип значения: «Число». Плановое время в пути в формате ч:мин

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Основные данные --> Адреса, маршруты, объекты строительства --> Маршруты".
