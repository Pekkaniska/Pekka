Справочник "Виды обслуживания ТС"

Назначение

Справочник "Виды обслуживания ТС" предназначен для классификации работ при проведении ТО, планово-профилактических работ, ремонтов и т.д.

Относится к подсистеме "Управление автотранспортом".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 25.

Не иерархический.

Реквизиты

  • Вид обслуживания ТС. Ссылается на перечисление «Виды обслуживания». Вид обслуживания ТС: Ввод в эксплуатацию, начисление амортизации, текущий ремонт, модернизация и т.п.
  • Описание. Тип значения «Строка». Произвольный комментарий к виду обслуживания.
  • Использовать в планировании ТО. Тип значения «Булево». Флаг устанавливает возможность использования вида обслуживания при планировании ремонтов (ТО).

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Ремонты --> Виды обслуживания ТС".
