Справочник "Тарифы ТС (на транспортные услуги)"

Назначение

Справочник содержит информацию о прейскурантах и тарифах, согласно которым ведется расчет стоимости оказанных транспортных услуг.

Относится к подсистеме "Работа ТС".

Реквизиты

  • Параметр выработки. Ссылается на справочник «Параметры выработки». Параметр выработки, по которому отбирается выработка водителей.
  • Минимальная выработка. Тип значения: «Число». Минимальная выработка в валюте упр. учета. Если указана, то при выработке меньше данного значения, тариф учитываться не будет.
  • Номенклатура услуги. Ссылается на справочник «Номенклатура». Номенклатура услуги, транспортные работы.
  • Сложный тариф. Тип значения: «Булево». Флаг "сложного" тарифа. "Сложным" считается тариф, размер которого определяется по совокупности условий, согласно таблице расчета (закладка "Настройка расчета").
  • Группировка тарифа 1-3. Ссылается на перечисление «Группировки тарифов». Группировка тарифа. Вид группировки: расстояние, время работы, грузоподъемность, класс груза.
  • Фиксированный тариф. Тип значения: «Булево». Флаг фиксированного тарифа. Фиксированным считается тариф, размер которого не зависит от количества выработки. Учитывается при расчете стоимости оказанных услуг транспортировки в документе «ТТД».
  • Тариф. Тип значения: «Число». Размер тарифа в валюте упр. учета. Используется только для "простого" тарифа.
  • Время начала / окончания. Тип значения: «Дата». Временной интервал действия тарифа.

Табличная часть «Группировки»

Табличная часть предназначена для настройки "сложных" тарифов (определение "сложного" тарифа см. выше). Настройка заключается в указании интервалов действия того или иного значения тарифа. Например, в зависимости от расстояния нужно определить следующие значения тарифа: до 100 км - 200 руб, от 100 до 200 км - 250 руб, свыше 200 км - 300 руб. Тогда нужно указать группировку 1 - "Расстояние" и ввести 3 строки: < 100 - 200, >= 100 И < 200 - 250, >= 200 - 300.

  • Значение группировки 1-3. Тип значения: «Число». Значение группировки 1.
  • Знак группировки 1-3. Ссылается на перечисление «Знаки сравнения». Знак группировки №1-3.
  • Тариф. Тип значения: «Число». Размер тарифа в валюте упр. учета. Используется для "сложных" тарифов.

Табличная часть «Область действия»

Табличная часть предназначена для настройки фильтров действия данного тарифа. Настройка выполняется с помощью стандартного механизма отборов (аналогичный механизм используется при настройке отборов в отчетах). Каждая строка отбора будет работать как условие "И". То есть, например, если указаны значения Контрагент, Тягач и Груз, то тариф будет применяться в контексте перевозки указанного груза указанным тягачем для указанного контрагента. Если требуется указать несколько значений (например, несколько контрагентов), на которые будет действовать тариф, необходимо их добавить одной строчкой отбора с видом сравнения "В списке" и в значении указать нужный список значений (контрагентов).

Формы

  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Доступен из справочника «Прейскуранты ТС».
