Справочник "Макеты ТС"

Назначение

Значения из этого справочника выбираются в форме элемента справочника «Модели ТС»  и используются в «АРМ механика» при выполнении операций по установке/замене шин на автомобилях.

Относится к подсистеме "Работа ТС".

Реквизиты

  • Хранилище макета. Тип значения «Хранилище значений». Рисунок макета ТС.
  • Дерево макета. Тип значения «Хранилище значений». Структура-дерево данных по макету ТС.
  • Хранить макет во внешнем файле.  Тип значения «Булево». Флаг хранения макета во внешнем файле. Если снят, то макет будет храниться непосредственно в информационной базе.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.

Способ вызова

Вызывается из меню "Основные данные --> Транспортные средства и оборудование --> Макеты транспортных средств".
