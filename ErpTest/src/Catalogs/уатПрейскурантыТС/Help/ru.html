Справочник "Прейскуранты ТС"

Назначение

Справочник содержит информацию по прейскурантам на транспортные услуги, согласно которым ведется расчет выработки ТС. Справочнику подчинен справочник «Тарифы ТС», в котором можно настроить область действия (фильтры) тарифа, а также таблицу "сложного" тарифа.

Установка действующих прейскурантов производится документом "Установка прейскурантов ТС".

Относится к подсистеме "Работа ТС".

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой введен прейскурант.
  • Тип прейскуранта. Тип значения «Число». 0 = для ТТД и ПЛ, 1 = для заказа на такси (УАТ ПП).
  • Дата начала. Тип значения «Дата». Начало действия прейскуранта.
  • Дата окончания. Тип значения «Дата». Окончание действия прейскуранта.
  • Валюта. Ссылается на справочник «Валюты». Валюта прейскуранта.
  • Сумма включает НДС. Тип значения «Булево». Флаг включения НДС в сумму.

Формы

  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Основные данные --> Прейскуранты на транспортные услуги".
