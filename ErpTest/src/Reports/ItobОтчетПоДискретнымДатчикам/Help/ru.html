Отчет по дискретным датчикам

Отчет предназначен для анализа данных, которые приходят с дискретных датчиков трекеров.

Отчет по дискретным датчикам можно вызвать из меню «Мониторинг» -> «Отчет по дискретным датчикам», или из контекстного меню обработки «Рабочее место диспетчера», при работе в режиме «Слежение» и используя таблицу детальных записей.
