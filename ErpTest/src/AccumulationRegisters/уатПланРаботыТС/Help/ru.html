Регистр накопления "План работы ТС"

Назначение

В этом регистре хранятся данные по планированию работ ТС.

Оборотный.

Измерения

  • Сценарий. Тип значения: справочник «Сценарии планирования». Сценарий планирования, описывающий периодичность, детализацию планирования. 
  • Подразделение. Тип значения: справочник «Подразделения организаций». Подразделение планирования. 
  • Детализация планирования. Тип значения: перечисление «Детализация планирования ТС». Имеет значения: Тип ТС, модель ТС, ТС. Дополнительное измерение для быстрой работы с регистром (детализация планирования уже указана в сценарии).
  • Документ планирования. Тип значения: документ «Планирование работы ТС». Документ ввода планов работы ТС. 
  • Номенклатура. Тип значения: справочник «Номенклатура». Номенклатура (грузы) планирования. 
  • Параметр выработки. Тип значения: справочник «Параметры выработки». Параметр выработки перевозимых грузов (время, расстояние перевозки).
  • Заказ. Тип значения: документ «Заказ на ТС». Документ по заказам на ТС.
  • Контрагент. Тип значения: справочник «Контрагенты». Заказчик.
  • Договор. Тип значения: справочник «Договоры контрагентов». Договор заказчика.

Ресурсы

  • Количество. Тип значения: «Число». Плановое количество перевозимого груза.
  • Стоимость. Тип значения: «Число». Плановая стоимость перевозки.
  • НДС. Тип значения: «Число». Плановая НДС.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Планирование работы ТС».
