Регистр накопления "Расход технических жидкостей на ТС"

Назначение

В этом регистре хранятся данные по расходу и приходу технических жидкостей на ТС.

Оборотный.

Измерения

  • Вид операции.
    • Тип значения: перечисление "Виды операций с техническими жидкостями". Вид операции с техническими жидкостями. 
  • Водитель. Тип значения: справочник «Сотрудники». Сотрудник, по которому ведется учет расхода технических жидкостей. 
  • Транспортное средство. Тип значения: справочник «Транспортные средства». ТС, по которому ведется учет расхода технических жидкостей. 
  • ГСМ. Тип значения: справочник «Номенклатура». Номенклатура технической жидкости. 
  • Организация. Тип значения: справочник «Организации». Организация, по которой ведется учет расхода технических жидкостей. 
  • Колонна. Тип значения: справочник «Колонны». Колонна, по которой ведется учет расхода технических жидкостей. 

Ресурсы

  • Количество расход. Тип значения: «Число». Расход технических жидкостей.
  • Количество приход. Тип значения: «Число». Приход (заправка) технических жидкостей.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Путевой лист».
