Документ "Товарно-транспортный документ"

Назначение

Документ «Товарно-транспортный документ» – аналог «бумажных» документов (ТТН и талоны заказчиков), которые отражают работу автомобиля у заказчика и заверяются им. Товарно–транспортные документы вводятся в таблицу ТТД путевого листа.

Табличная часть закладки «Выработка» заполняется автоматически на основании данных закладки «Задание» путевого листа. Так же возможно заполнить по кнопке «Заполнить выработку по выбранному ПЛ» и «Заполнить выработку в строке по ПЛ». При нажатии кнопки «Заполнить выработку по выбранному ПЛ»,  заполняется выработка по заданиям выбранного путевого листа. При нажатии кнопки «Заполнить выработку в строке по ПЛ», заполнится строка выработки по заданиям путевого листа.  

Закладка «Стоимость» содержит расчет стоимости транспортных услуг. Табличные данные заполняются по кнопке «Рассчитать». Рассчитанные данные можно отредактировать или ввести вручную. При расчете используется справочник «Прейскуранты на транспортные услуги».

Закладка «ТТН» содержит реквизиты, которые используются при печати формы «ТТН».

Если для данного документа были сформированы соответствующие документы типовых конфигураций 1С, то такой документ уже нельзя изменить. Для возможности внесения изменений необходимо удалить сформированные типовые документы. Это может сделать только пользователь с правами «Администратор управления автотранспортом» из меню «Сервис --> Регламентные операции --> Сформированные документы типовой конфигурации». Необходимо найти нужный документ УАТ и нажать клавишу Del. Все сформированные для него документы типовой конфигурации будут помечены на удаление.

Реквизиты

  • Отразить в управленческом/бухгалтерском/налоговом учете. Признак использования документа в определенном виде учета. Используется при работе с типовыми документами.
  • Организация. Ссылается на справочник «Организации». 
  • Контрагент. Ссылается на справочник «Контрагенты». 
  • Договор контрагента. Ссылается на справочник «Договоры контрагентов». 
  • Номер ТТН по учету контрагента. Тип значения: «Строка». Номер документа ТТН контрагента.
  • Номер пачки. Тип значения: «Число». Номер пачки (если ведется учет по пачкам).
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за заполнение документа. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий. 

Табличная часть «Выработка»

  • Путевой лист. Ссылается на документ «Путевой лист». Путевой лист, по которому введен данный ТТД. 
  • Маршрут. Ссылается на справочник «Маршруты». Маршрут движения.
  • Груз. Ссылается на справочник «Номенклатура». Перевозимый груз.
  • Подача. Тип значения: «Дата».  Время подачи, в часах и минутах.
  • Время работы. Тип значения: «Дата».  Время в работе, в часах и минутах.
  • Количество ездок. Тип значения: «Число». Количество ездок. 
  • Количество операций. Тип значения: «Число». Количество операций. 
  • Перевезено за ездку. Тип значения: «Число».  Вес груза, перевезенный за одну ездку, в тоннах.
  • Пробег с грузом за ездку. Тип значения: «Число». Пробег за одну ездку с грузом, в км.
  • Пробег порожний за ездку. Тип значения: «Число». Пробег за одну ездку порожний, в км.
  • Погрузочно-разгрузочные работы. Тип значения: «Число». Время, вес или работа по погрузке/разгрузке, в часах, тонночасах или тоннах.
  • Простой под погрузкой. Тип значения: «Число». Время простоя при погрузке, в часах и минутах. 
  • Простой под разгрузкой. Тип значения: «Число». Время простоя при разгрузке, в часах и минутах. 
  • Пробег специальной работы. Тип значения: «Число». Пробег специальной работы (может быть введено до 5 различных видов спец. работ), в км.
  • Выработка, Параметр выработки. Здесь вводится произвольный параметр выработки (в справочнике Параметры выработки можно создавать дополнительные пользовательские параметры выработки) и его значение.

Табличная часть «Стоимость»

  • Путевой лист. Ссылается на документ «Путевой лист». Путевой лист, по которому введен данный ТТД.
  • Маршрут. Ссылается на справочник «Маршруты». Маршрут движения.
  • Объект строительства. Ссылается на справочник «Объекты строительства». Объект строительства, по которому выполняется задание.
  • Вид перевозки. Ссылается на справочник «Виды перевозок». Вид транспортной услуги.
  • Груз. Ссылается на справочник «Номенклатура». Перевозимый груз.
  • Параметр выработки. Ссылается на справочник «Параметры выработки ТС и водителей». 
  • Количество параметра выработки. Тип значения: «Число». Количество по параметру выработки.
  • Услуга. Ссылается на справочник «Номенклатура». Номенклатура услуги по перевозке груза. 
  • Количество. Тип значения: «Число». Количество транспортных услуг. 
  • Цена. Тип значения: «Число». Цена транспортных услуг.
  • Сумма. Тип значения: «Число». Сумма транспортных услуг. 

Закладка «ТТН»

На закладке находятся дополнительные реквизиты и реквизиты, предназначенные для вывода на печать товарно-транспортной накладной:

  • Грузоотправитель. Ссылается на справочник «Контрагенты». Контрагент, отправляющий груз.
  • Грузополучатель. Ссылается на справочник «Контрагенты». Контрагент, получающий груз.
  • Пункт погрузки. Тип значения: «Строка», ссылается на справочник «Пункты назначения». Адрес отправления груза.
  • Пункт разгрузки. Тип значения: «Строка», ссылается на справочник «Пункты назначения». Адрес доставки груза.
  • Переадресовка. Тип значения: «Строка», ссылается на справочник «Пункты назначения». Адрес переадресовки груза. Адрес пункта, куда необходимо перенаправить груз в момент его доставки в пункт назначения.
  • Вид перевозки. Ссылается на справочник «Виды перевозок». Вид перевозок.
  • Табличная часть "ТТН". Это расширение табличной части "Выработка". Поля используются для вывода на печать ТТН.

Способ вызова

Вызывается из меню "Работа ТС --> Услуги --> ТТД"

Движения

  • «Выработка по стоимости». Выработка по стоимости (обороты).
