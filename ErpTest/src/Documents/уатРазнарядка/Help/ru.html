Документ "Разнарядка"

Данный документ относится к подсистеме "Управление автотранспортом".

Реквизиты шапки документа

  • Организация. Организация, оказывающая услуги предоставления автотранспорта и строительной техники.
  • Колонна. Реквизит необязателен для заполнения. Если он заполнен, то при нажатии кнопки "Заполнить" разнарядка будет заполнена  только по транспортным средствам данной колонны.
  • Подразделение. Реквизит необязателен для заполнения. Если он заполнен, то при нажатии кнопки "Заполнить" разнарядка будет заполнена  только по транспортным средствам данного подразделения.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.

Заполнение табличной части документа

Табличная часть может быть по желанию пользователя заполнена автоматически на произвольно выбранную дату по кнопке "Заполнить". Данные по транспортным средствам, режимы работы, время выезда и время возвращения заполняются из регистра сведений "Первоначальные данные ТС". Данные по водителям/кондукторам берутся из регистра сведений "Экипажи ТС". При этом, по регистру сведений "Графики работы по видам времени", проверяется, работают ли члены экипажа по графику на текущий день. Если работающих членов экипажа на текущий день обнаружено не будет, то будет подставлен основной водитель и первый не основной водитель. Контрагент, адрес прибытия, время прибытия и время убытия автоматически не проставляются.

Печать

По кнопке печать распечатывается Разнарядка на выпуск ТС.

 

Проведение документа

Документ производит движения по регистру сведений "Состояние ТС".
