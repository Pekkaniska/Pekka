Списание товаров с хранения

Документ предназначен для отражения факта списания принятых на хранение товаров у поклажедателя, в случае если данные товары были списаны по результатам инвентаризации и с поклажедателем установлена договоренность о списании товаров за его счет.

Возможность использования в системе документов приемки на ответственное хранение определяется функциональной опцией Приемка на ответственное хранение с правом продажи в разделе НСИ и администрирование - Закупки - Документы закупок.

Оформляется вручную, может быть оформлен из помощника оформления складских актов по результатам инвентаризации.

При проведении документа, контролируется количество принятых товаров на ответственное на хранение у указанного поклажедателя по указанному договору приемки. Документ не будет проведен, если на указанном месте хранения нет достаточного количества принятых товаров.

В результате проведения уменьшается количество принятых товаров на хранение, кредиторская задолженность перед поклажедателем не возникает.

Основное

На закладке заполняется информация о владельце товаров (поклажедателе), который передал товары на ответственное хранение, оформленные при помощи документа Приемка товаров на хранение, заполняются данные о месте хранения и указывается тип операции списания.

Типы операций списания:

  • Списание товаров, хранящихся на складе (в качестве места хранения можно указать только склад);
  • Списание товаров, переданных в производство (в качестве места хранения можно указать только цеховую кладовую).

Товары

Заполняются данные о списываемых товарах, есть возможность автоматического заполнения по данным товаров принятых на ответственное хранение.
