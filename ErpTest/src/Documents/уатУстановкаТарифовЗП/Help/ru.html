Документ "Установка тарифов зарплаты сотрудников"

Назначение

Документ служит установки действующих тарифов и тарифных планов начисления зарплаты сотрудников.

Относится к подсистеме "Работа ТС".

Параметры

Длина номера: 11.

Периодичность номера: Непериодический.

Тип номера: Строка.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация предприятия, от имени которой оформляется документ.
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ. 

Табличная часть «Тарифы»

  • Тариф. Ссылается на справочники «Тарифы з/п», «Тарифные планы з/п». Тариф или тарифный план начисления зарплаты.
  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник, для которого устанавливается тариф.
  • Дата начала. Тип значения: «Дата». Дата начала периода действия тарифа.
  • ДатаОкончания. Тип значения: «Дата». Дата, до которой действует тариф. 

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "Управление персоналом --> Учет зарплаты --> Установка тарифов ЗП".

Движения

  • «Тарифы сотрудников». Устанавливает начало периода действия тарифа для начисления зарплаты сотруднику.
