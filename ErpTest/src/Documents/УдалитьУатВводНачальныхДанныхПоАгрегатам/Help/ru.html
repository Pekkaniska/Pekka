Документ "Ввод начальных данных по агрегатам"

Назначение

Документ предназначен для ввода начальных данных по шинам, аккумуляторам и другим агрегатам на транспортные средства. Для шин имеется возможность ввести начальный пробег и износ.

Реквизиты

  • Организация. Организация, в которой числятся основные средства, указанные в табличной части.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.
  • МОЛ. Сотрудник, который несет полную материальную ответственность за ущерб, возникший в результате необеспечения сохранности вверенных ему под отчет агрегатов.

Табличная часть «Шины»

  • Серия номенклатуры. Шина, по позиции которого вводятся начальные остатки.
  • ТС. Транспортное средство, на которое вводим начальные данные по агрегатам.
  • Состояние. Используемые значения - "Установлено в работе", "Установлено в запас" и "Снято".
  • Место установки. Место установки шины. Например, "Переднее левое", "Заднее правое внутреннее".
  • Стоимость. Стоимость шины.
  • Пробег. Начальный пробег, км.
  • Износ. Начисленный износ.

Табличная часть «Аккумуляторы»

  • Серия номенклатуры. Аккумулятор, по позиции которого вводятся начальные остатки.
  • ТС. Транспортное средство, на которое вводим начальные данные по агрегатам.
  • Состояние. Используемые значения - "Установлено в работе" и "Снято".
  • Начало эксплуатации. Тип значения: «Число». Пробег начальный, км.

Табличная часть «Прочие агрегаты»

  • Серия номенклатуры. Шина или аккумулятор, по позиции которого вводятся начальные остатки.
  • ТС. Транспортное средство, на которое вводим начальные данные по агрегатам.
  • Годен до. Дата до которой годен агрегат (аптечка, огнетушитель).

Относится к подсистеме "Учет агрегатов".
