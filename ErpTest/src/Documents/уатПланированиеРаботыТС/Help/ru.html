Документ "Планирование работы ТС"

Назначение

Документ предназначен для отражения в управленческом учете следующих операций:

  • Ввод в систему и хранение данных по предполагаемым объемам работы ТС
  • Формирование планов работы ТС по данным подсистемы планирования и фактическим данным смежных подсистем

Ввод в систему и хранение данных по предполагаемым объемам работы ТС

 

В документе может указываться оценка предполагаемых объемов работы ТС или для ТС (уточненное планирование),  или для модели ТС,  или типа ТС (укрупненное планирование). Оценка объема работ указывается для отдельного периода по отдельному сценарию.

Реквизиты «Сценарий», «Дата планирования» являются обязательными. Заполнение реквизита «Сценарий» определяет:

  • Периодичность планирования, используемую в данном документе;
  • Детализация планирования – ТС, модели ТС, Тип ТС.

Реквизит «Дата планирования» выравнивается по периодичности планирования (на начальную границу).

Состав плановых данных вводится в табличную часть документа.  По каждому объекту указывается плановый объем работы ТС в количественной (если по сценарию ведется планирование количества работ) оценке и суммовой (если по сценарию ведется планирование сумм работ) оценке с выделением НДС.

Валюта документа, тип цен, признаки учета НДС настраиваются штатным для документов образом – в диалоговом окне «Цены и валюта» документа.

 

Формирование планов закупок по данным подсистемы планирования и фактическим данным смежных подсистем


Состав документа может быть заполнен в автоматическом режиме, для этого используется форма заполнения, вызываемая по кнопке «Заполнить».

В форме заполнения производится настройка правил заполнения документов.

Заполнение информации производится на основании заказов на ТС и (или) на основании выработки ТС за указанный период. Также возможно заполнение на основании выручки по путевым листам.

Для записанного документа возможно формирование печатной формы.

Документ не используется для ведения регламентированного учета.

Относится к подсистеме "Управление автотранспортом".

Способ вызова

Вызывается из меню "Планирование --> Планирование работы ТС"
