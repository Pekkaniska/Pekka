Подбор по остаткам

Форма предназначена для быстрого отбора товаров в документ корректировки аналитики обособленного учета.

  • Если в документе предварительно установлен вид аналитики и его конкретное значение, то при открытии формы автоматически будут заполнены соответствующие фильтры и будет установлен отбор товара.
  • По отобранному товару показывается доступное количество по выбранной аналитике.
  • После отбора необходимо отметить флажками те позиции которые будут перенесены в документ, а так же установить требуемое для корректировки количество товара.
  • По команде Перенести в документ отмеченные позиции будут добавлены в документ корректировки.
