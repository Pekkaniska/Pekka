Рабочее место

В Рабочем месте отображется список невыполненных распоряжений, по которым необходимо оформить документы или не завершен процесс отгрузки (приемки) по ордерной схеме.

Для индикации состояния оформления документов ордеров, оформленных по распоряжениям, предусмотрены две информационные колонки с пиктограммами. Пояснение назначения пиктограмм приведено в соответствующих колонкам отборах.

По команде Оформить по распоряжениям создается документ и заполняется всеми позициями распоряжений, по которым не оформлены документы. По команде Оформить по ордерам создается документ движения и заполняется всеми отгруженными (принятыми) по ордерам позициями.
