Документ "ДТП"

Назначение

Документ служит для регистрации дорожно-транспортных происшествий.

Относится к подсистеме "Учет агрегатов".

Основные реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой зафиксировано ДТП.
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ. 
  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство организации, попавший в ДТП. 
  • Водитель. Ссылается на справочник «Сотрудники». Водитель, попавший в ДТП. 
  • Место ДТП
    • . Тип значения: «Строка». Место, где произошло ДТП.
  • Причина ДТП. Ссылается на справочник «Причины ДТП». Причина, по которой произошло ДТП.
  • Виновность. Тип значения: «Булево». Флаг, определяющий виновность водителя.
  • Сумма ремонта. Тип значения: «Число». Стоимость ремонта.

Реквизиты данных ГИБДД

  • Номер справки. Тип значения: «Строка». Номер справки ГИБДД.
  • Дата справки. Тип значения: «Дата». Дата выписки справки ГИБДД.
  • Повреждения. Тип значения: «Строка». Описание повреждений ТС в результате ДТП.
  • Принятые меры. Тип значения: «Строка». Описание принятых мер по ДТП.

Реквизиты независимой экспертизы

  • Контрагент. Ссылается на справочник «Контрагенты». Независимый эксперт.
  • Номер. Тип значения: «Строка». Номер решения экспертизы.
  • Дата. Тип значения: «Дата». Дата решения экспертизы.
  • Сумма ущерба. Тип значения: «Число». Сумма ущерба, оценка экспертизы.
  • Примечание. Тип значения: «Строка». Примечание экспертизы.

Реквизиты страховой компании

  • Страховая компания. Ссылается на справочник «Контрагенты». Страховая компания.
  • Номер дела. Тип значения: «Строка». Номер дела страховой компании.
  • Дата. Тип значения: «Дата». Дата рассмотрения дела страховой компании.
  • Сумма выплаченная. Тип значения: «Число». Сумма, выплаченная страховой компанией.
  • Примечание. Тип значения: «Строка». Примечание страховой компании.

Табличная часть «Участники»

  • Автомобиль. Тип значения: «Строка». Автомобиль, участвовавший в ДТП. Указываются другие ТС (помимо основного нашего), попавшие в ДТП.
  • Водитель. Тип значения: «Строка». Водитель автомобиля, участвовавшего в ДТП.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Табличная часть «Ремонты»

На этой закладке указываются список ремонтных листов, по которым выполнялся ремонт автомобиля.

Способ вызова

Вызывается из раздела "Управление ремонтами и учет агрегатов / ДТП / Дорожно-транспортные происшествия"
