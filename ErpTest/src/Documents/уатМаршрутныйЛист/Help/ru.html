Документ "Маршрутный лист"

Документ служит для формирования и регистрации маршрута транспортного средства по развозке грузов. Формирование маршрута может осуществляться на основании введенных ранее заказов на ТС (операция "Заполнение маршрута по заказам на ТС") или формирования произвольного списка развозки грузов (операция "Ручное заполнение табличной части").  В документе представляется маршрут транспортного средства с детализацией по грузам.

В случае формирования маршрутного листа по заказам на ТС в табличную часть документа можно добавить только неразвезенные грузы по ранее введенным заказам и произвольные пункты назначения. Маршрут в данном случае представляется в виде дерева, в котором отображаются следующие параметры:

  • время и расстояние до следующего пункта маршрута;
  • планируемое время выполнения операция по заказу и суммарное время стоянки в пункте;
  • планируемое и фактическое времена прибытия/убытия из пункта;
  • временное окно доступности пункта;
  • операция выполняемая в данном пункте по заказу;
  • параметры грузов;
Пользователь может менять порядок следования пунктов посредством перетаскивания. При этом необходимо помнить, что перетаскивание, в результате которого пункт разгрузки окажется выше соответствующего ему пункта погрузки невозможно. При проведении документа с данной операцией производится контроль остатков по необработанным заказам на ТС.

При ручном формировании маршрутного листа в маршрут могут быть добавлены произвольные пункты и грузы. Контроль остатков при проведении в данном случае не выполняется.

Если конфигурация используется в объединении с конфигурацией "Транспортная логистика и экспедирование", то в случае перевозки собственными ТС перевозчик выбирается из списка Организаций, в случае же перевозки привлеченными ТС перевозчик выбирается из списка контрагентов.
