Документ "Ввод остатков ГСМ"

Назначение

Документ служит для учета остатков ГСМ, которые были в транспортных средствах на момент начала работы с программой.

Относится к подсистеме "Учет ГСМ".

Параметры

Длина номера: 11.

Периодичность номера: Непериодический.

Тип номера: Строка.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой вводятся остатки ГСМ. 
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ. 

Табличная часть «Топливо»

  • ТС. Ссылается на справочники «Транспортные средства». Транспортное средство или оборудование.
  • ГСМ. Ссылается на справочник «Номенклатура». Номенклатура топлива. 
  • Количество. Тип значения: «Число». Количество вводимого топлива. 
  • Сумма. Тип значения: «Число». Стоимость вводимого топлива.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из меню "ГСМ --> Ввод остатков ГСМ"

Движения

  • «Остатки ГСМ в ТС». Отражаются партии остатков ГСМ на ТС. Документ выполняет приход по данному регистру.
