Регистр сведений "Местоположение ТС по GPS"

Назначение

Регистр предназначен для хранения истории местоположения транспортных средств по данным системы спутникового мониторинга (GPS).

Измерения

  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство.

Ресурсы

  • Лат. Широта пункта назначения.
  • Лон. Долгота пункта назначения.
  • Скорость. Текущая скорость ТС.
  • Пункт. Представление пункта местоположения.
  • Значение одометра. Накопленное значение показаний одометра в километрах.
  • Потребленное топливо. Потребленное топливо в сантилитрах.
  • Уровень топлива в баке. Уровень топлива в баке (%).
  • Значение датчика 1,2,3,4,5. Значения датчиков, дополнительно установленных на ТС. 

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Способ вызова

Вызывается из меню "Работа ТС --> Мониторинг --> Местоположение ТС по GPS".
