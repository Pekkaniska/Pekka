Регистр сведений "Дополнительные сведения ТС по GPS"

Назначение

Регистр предназначен для хранения дополнительных сведений транспортных средств по данным системы спутникового мониторинга (GPS).

Содержит записи о следующих видах событий: заправка ГСМ, слив ГСМ, мгновенное превышение скорости, долгосрочное превышение скорости, нарушение предельных значений датчиков.

Параметры

Периодичность: Периодический, в пределах секунды.

Режим записи: Независимый.

Способ вызова

Вызывается из меню "GPS --> Дополнительные сведения по GPS".
