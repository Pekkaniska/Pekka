Счета регл. учета, игнорируемые при отражении в международном учете

Всем счетам регламентированного учета должно быть указано соответствие счетам международного учета, если счету не будет обнаружено соответствие, то документы с такими проводками будут ожидать уточнения настроек.
Для игнорирования некоторых счетов для механизма трансляции операций из регламентированного учета, счета могут быть помечены как игнорируемые при отражении.
